\doxysubsection{Detailed Description}
Structure representing a deque with PID. 

This structure contains metadata about the deque with Process ID (PID), including the number of elements, a pointer to the beginning of the deque ({\ttfamily front}), and a pointer to the end of the deque ({\ttfamily back}).
